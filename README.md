# Data-Driven-Programming-with-CAS-in-SAS-Viya

Want to process every file in a directory?  Want to limit which get processed according to an adjustable list?  How about processing each differently according to fields on a table?  Let’s explore data driven processing in Viya® .  From the SAS®  Macro  language to PROC CAS’ own language, CASL, we’ll show you how to drive your processing from the data it’s running on.
With the adoption of SAS® Viya®, processing in CAS is becoming more common, and CAS speaks CASL. Seasoned SAS® coders often use SAS® macro to produce data-driven programs, automating tedious programming tasks. When working with CAS from the SAS®  Compute Server, it's important to know where, when, and how the CASL that executes is generated. Whatever your experience level, the interactions between SAS® code, CASL, and Macro can be intimidating. This presentation aims to demystify that process.
